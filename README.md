# KaKafka (Kinda Almost Kafka)

Use the following:
* java 17
* [maven wrapper](https://maven.apache.org/wrapper/index.html) with maven 3.9.3 
`mvn wrapper:wrapper -Dmaven=3.9.3`
* [Netty 4.x](https://netty.io/wiki/user-guide-for-4.x.html)

## Build
```
./mvnw clean package
```

## Unit tests.

```
./mvnw test
```
